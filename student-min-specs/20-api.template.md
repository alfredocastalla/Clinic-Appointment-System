# API Specification

## Authentication
- POST /auth/register/user
- POST /auth/register/doctor
- POST /auth/login

## Users
- GET /users
- GET /users/:id
- PATCH /users/:id
- DELETE /users/:id

## Doctors
- GET /doctors
- GET /doctors/:id

## Appointments
- POST /appointments
- GET /appointments
- PATCH /appointments/:id/cancel
