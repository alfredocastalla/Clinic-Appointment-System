# Context

## System Overview
The Clinic Appointment System is a web-based platform that allows patients to book appointments with doctors while enabling doctors to manage their schedules efficiently.

## Goals
- Simplify appointment booking
- Reduce manual scheduling errors
- Improve doctor-patient coordination

## Stakeholders
- Patients (Users)
- Doctors
- Admin (future role)

## Assumptions
- Users have internet access
- Doctors manage their own availability
- System is used in a clinic environment
